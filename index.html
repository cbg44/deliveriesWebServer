Deliveries Web-Service API

Use the methods below in order to fetch data from JSON

With this API you'll be able to get all the deliveries saved in DB,
get specific delivery by posting unique ID, and get a list of deliveries specified by their rally point type.

Documentation

Get all Deliveries

Method: GET
Request URL: https://deliveriesws.herokuapp.com/getAllDeliveries
Response: JSON of all deliveries from DB.

Get Delivery By ID

Method: POST
Required Parameters: delivery[id] = {id}
Request URL: https://deliveriesws.herokuapp.com/getDeliveryByID
Response: JSON of single delivery identified by the ID.
{
    deliveries: [
        {
              "delivery": {
                "id": 123456,
                "properties": {
